# 📚 Manual Básico de Markdown

Este proyecto contiene una guía sencilla sobre el uso de **Markdown**, un lenguaje de marcado ligero utilizado para crear documentación estructurada.

Markdown es ampliamente utilizado en plataformas como GitHub para crear archivos de documentación como README, manuales técnicos y guías de proyectos.

---

# 🎯 Objetivo

Explicar de manera simple cómo utilizar Markdown para crear documentos con:

- Títulos
- Listas
- Tablas
- Enlaces
- Código

Además, mostrar cómo realizar **commits desde la terminal usando Git**.

---

# 🛠 Tecnologías utilizadas

- Markdown
- Git
- Terminal de comandos

# 🚀 Cómo hacer un commit desde la terminal

1. Verificar estado del repositorio

```bash
git status
```

2. Agregar archivos

```bash
git add .
```

3. Crear commit

```bash
git commit -m "Agregando documentación en Markdown"
```

4. Subir cambios al repositorio

```bash
git push origin main
```
